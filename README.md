## Kullanım
1. Firmaları girin.
2. Ürünleri ve renk seçeneklerini girin.
3. Yeni satış/sipariş oluşturun.
4. Siparişler ekranından durum, teslimat ve ödeme takibi yapın.
5. Yedek/Ayarlar ekranından Excel ve veritabanı yedeği alın.
